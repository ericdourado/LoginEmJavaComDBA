## TELA DE LOGIN EM JAVA/SQL 



- Projeto realizado em criar uma tela de login onde o usuário realiza o cadastro em um ambiente grafico (SWING) e é armazenado no banco de dados (postgresql).
- Para fazer login, é realizado uma consulta dentro do banco de dados para saber se o cliente/user existe.
- Foi utilizado métodos MVC para a criação do projeto.

- OBS: O BACKUP DO BANCO ESTÁ NO ARQUIVO COM O NOME usersJava.sql
